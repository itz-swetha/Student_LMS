Student LMS – MongoDB Schema Validation Experiments
Overview

This project demonstrates the use of MongoDB schema validation in a Student Learning Management System (LMS).
It includes multiple experiments to understand how collections are created, validated, and modified.

Tech Stack
MongoDB
MongoDB Shell (mongosh)
JSON Schema Validation

Database Used
lmsDB

Key Learnings
How to create and drop collections
Implement schema validation in MongoDB

Use of:
required
bsonType
pattern
enum
min/max
Modify existing collections using collMod
Verify schema using getCollectionInfos()
